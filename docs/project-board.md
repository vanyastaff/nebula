# Project Board

The project board tracks progress from idea to completion.

## Board Flow

1. Backlog
2. Ready
3. In Progress
4. In Review
5. Done

## Operational Rules

- Every in-progress item should have an assignee.
- Keep card status synchronized with PR status.
- Link board items to issues and PRs for traceability.
