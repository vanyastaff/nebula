# Issues

Issue tracking is the primary planning and coordination interface.

## Issue Types

- Bug reports
- Feature requests
- Documentation updates
- Refactors and maintenance tasks

## Triage Guidelines

- Reproduce and scope bugs before assigning.
- Label by area, priority, and difficulty.
- Link related PRs and roadmap tasks.
