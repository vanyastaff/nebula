# EventBus Integration Examples

This folder provides practical examples for integrating `nebula-eventbus` with runtime domains.

## Example Set (T025)

- `log-events.md` - bridge eventbus streams into structured logs.
- `metrics-events.md` - record `EventBusStats` snapshots using `nebula-metrics`.
- `dashboard-events.md` - build a lightweight in-memory dashboard projection from filtered/scoped subscriptions.

## Notes

- Examples are intentionally lightweight and documentation-first.
- They focus on transport semantics and integration patterns, not on distributed backends.
