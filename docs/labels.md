# Labels

Labels define issue semantics and improve triage speed.

## Recommended Label Dimensions

- Type: bug, feature, docs, chore
- Priority: high, medium, low
- Scope: crate or subsystem
- Difficulty: good-first-issue, medium, hard

## Usage Rules

- Apply at least one type and one scope label.
- Keep priority labels limited to active queue items.
- Revisit stale labels during grooming.
