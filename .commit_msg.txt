refactor(action): remove ActionType, ExecutionMode, RetryPolicy, TimeoutPolicy

Remove policy and type discriminant types from metadata as part of
simplifying the action system.

- ActionType, ExecutionMode: removed enums
- RetryPolicy, TimeoutPolicy: removed structs and builder methods
- DeferredOutput.retry: replaced with DeferredRetryConfig in output.rs
- Macros: removed action_type and isolation attribute parsing
- Tests: updated metadata tests, macro UI tests
